# 🎮 Alphabet Collision Game

A C++ and SFML based typing game where players eliminate falling letters by pressing the correct keys, featuring real-time scoring based on speed and accuracy.

---

## 🎯 Game Overview

The Alphabet Collision Game is an interactive typing game developed using **C++** and the **SFML** library. Alphabet letters fall randomly from the top of the screen, and players must press the corresponding keys to eliminate them before they reach the bottom.

---

## ✨ Features

- 🎲 Randomly generated falling letters
- ⌨️ Real-time keyboard input detection
- 📊 Score based on speed and accuracy
- 📈 Increasing difficulty levels
- 🎨 Interactive GUI using SFML
- 💀 Game-over with score tracking

---

## 🛠️ Tech Stack

| Category | Technologies |
|----------|--------------|
| Language | C++ |
| Graphics | SFML |
| IDE | Visual Studio / CodeBlocks |

---

## 🎮 How to Play

1. Letters fall from the top
2. Press the matching key on your keyboard
3. Correct keys eliminate letters and add points
4. Don't let letters reach the bottom!
5. Game ends when lives reach zero

## 🚀 Installation

### Prerequisites
- C++ Compiler (GCC/MinGW/MSVC)
- SFML Library

### Steps

1. **Clone repository**
   ```bash
   git clone https://github.com/Imbisat-Mahmood/alphabet-Collision-Game.git
   ```

2. **Install SFML**
   - Windows: Download from [SFML Website](https://www.sfml-dev.org/)
   - Linux: `sudo apt-get install libsfml-dev`

3. **Compile**
   ```bash
   g++ GAME.cpp -o game -lsfml-graphics -lsfml-window -lsfml-system
   ```

4. **Run**
   ```bash
   ./game
   ```
